# MongoDB_MetaSearch

MetaSearch tool using MongoDB and LibreCat/Catmandu

== Dependencies 

PHP

MongoDB - 3.2.1 or later (http://www.mongodb.org/)

Librecat/Catmandu (http://librecat.org/Catmandu/)

== Install == 

Git clone 

Edit inc/config.php

Import data with Librecat/Catmandu
